# Twitch Desktop

App desktop leve que abre o [Twitch](https://www.twitch.tv) em uma janela limpa, sem abas de navegador.

- ✅ **Login automático** — usa seus cookies do navegador, sem precisar logar de novo
- ✅ **Multiplataforma** — Windows, Linux e macOS
- ✅ **20+ navegadores** — Chrome, Edge, Brave, Firefox, Opera, Vivaldi e mais
- ✅ **Código aberto** — Python puro, ~10 MB
- ✅ **Sem telemetria** — não coleta nenhum dado

## Download

| Plataforma | Download |
|------------|----------|
| **Windows** | [`Twitch-Setup.exe`](https://github.com/Ryanabcraft/twitch-desktop/releases/latest) — Instalador Inno Setup |
| **Linux / macOS** | [`twitch_app.py`](twitch_app.py) — Rode com `python3 twitch_app.py` |

## Como usar

### Windows
1. Baixe o instalador `.exe` da [última release](https://github.com/Ryanabcraft/twitch-desktop/releases/latest)
2. Execute e siga os passos
3. Pronto! O Twitch abre em janela limpa

### Linux / macOS
```bash
wget https://raw.githubusercontent.com/Ryanabcraft/twitch-desktop/master/twitch_app.py
python3 twitch_app.py
```

Requer Python 3.6+ e um navegador instalado.
